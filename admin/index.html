📋 Como usar este painel

1. Selecione a seção no menu lateral (ex: TN → Corridas)
2. Preencha o formulário e clique em Salvar
3. Vá em Exportar / Baixar para baixar os JSONs atualizados
4. Faça upload dos arquivos JSON no repositório GitHub na pasta data/
5. O site atualiza automaticamente em ~1 minuto via GitHub Actions

⚠️ Importante

As alterações feitas aqui ficam salvas apenas nesta sessão do navegador.
Para publicar no site, você precisa baixar os JSONs e fazer upload no GitHub.

Dica: abra este painel no mesmo navegador que você usa para o GitHub, assim o fluxo fica mais rápido.
